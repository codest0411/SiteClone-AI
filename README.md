│

## 🚀 Getting Started

<br/>

### Prerequisites

```
✓  Google Chrome 114 or higher   (Side Panel API support)
✓  A free Groq API key           →  https://console.groq.com
✓  That's it — no Node.js, no build step, no bundler needed.
```

<br/>

### Installation — 4 Steps

```bash
# ── Step 1 ── Clone the repo ──────────────────────────────────────
git clone https://github.com/codest0411/SiteClone-AI.git
cd SiteClone-AI

# ── Step 2 ── Add your Groq API key ───────────────────────────────
# Open background.js and set your key on line 1:
const GROQ_API_KEY = "gsk_xxxxxxxxxxxxxxxxxxxx";
# Get yours FREE at: https://console.groq.com

# ── Step 3 ── Load in Chrome ───────────────────────────────────────
# → Open Chrome and go to: chrome://extensions
# → Enable "Developer Mode" toggle (top-right corner)
# → Click "Load unpacked"
# → Select the SiteClone-AI folder

# ── Step 4 ── Use it! ──────────────────────────────────────────────
# → Visit any website (try adidas.com, stripe.com, notion.so)
# → Click the SiteClone AI icon in your Chrome toolbar
# → Hit "Analyze This Site"
# → Wait ~3–5 seconds for AI analysis
# → Copy your prompt → paste into Claude / ChatGPT / Cursor
# → Start building! 🚀
```

## 🐛 Known Issues & Fixes

<br/>

| Issue | Root Cause | Fix / Workaround |
|-------|-----------|------------------|
| "Could not analyze this page" | Site's CSP blocks content script injection | Hard-refresh the page (`Ctrl+Shift+R`), then try again |
| Colors extracted look incorrect | Dynamic styles not loaded at extraction time | Wait 2–3 seconds after full page load, then analyze |
| Side panel doesn't open at all | Chrome version below 114 | Update Chrome to latest stable version |
| "Invalid API key" error | Key not saved / typo in key | Double-check key in `background.js`, reload extension |
| Response takes 10+ seconds | Groq server under load | Wait a moment and click "Refresh" to retry |
| Missing components in output | Site uses heavily obfuscated class names | Partial result expected — prompt will still be useful |
| Prompt shows "unknown" tech stack | No detectable framework fingerprints | Site may use a custom/vanilla setup — still generates correctly |

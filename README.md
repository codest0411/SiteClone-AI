## рҹҡҖ Getting Started

<br/>

### Prerequisites

```bash
# You need:
вң“ Google Chrome 114+  (Side Panel API support)
вң“ A free Groq API key  вҶ’  https://console.groq.com
вң“ That's it. No Node.js, no build step.
```

<br/>

### Installation

```bash
# Step 1 вҖ” Clone the repo
git clone https://github.com/yourusername/siteblueprint-ai
cd siteblueprint-ai

# Step 2 вҖ” Add your Groq API key
# Open background.js and replace line 1:
const GROQ_API_KEY = "your_groq_api_key_here";

# Step 3 вҖ” Load in Chrome
# вҶ’ Go to chrome://extensions
# вҶ’ Enable "Developer Mode" (top right toggle)
# вҶ’ Click "Load unpacked"
# вҶ’ Select the siteblueprint-ai folder

# Step 4 вҖ” Use it!
# вҶ’ Visit any website (e.g. adidas.com)
# вҶ’ Click the SiteBlueprint AI icon in toolbar
# вҶ’ Click "Analyze This Site"
# вҶ’ Copy your prompt and paste into Claude / ChatGPT
```
